# WALLET SCAN
*Software tool to analyze your crypto wallet. Available in 9 chains*

## Instructions
1. Get api key from https://goldrush.dev/
2. Use name GOLD_RUSH_API_KEY
3. Install streamlit
  ```bash
  pip install streamlit
  ```
4. Run project:
  ```bash
  streamlit run main.py
  ```
